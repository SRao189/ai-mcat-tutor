## Definitions

### Br√łnsted-Lowry (Most important for MCAT)
- **Acid** = proton (H‚Āļ) donor
- **Base** = proton (H‚Āļ) acceptor

### Lewis (Broader)

- **Base** = electron-pair donor

**Example (heme group):** O‚āā binds to Fe¬≤‚Āļ
- O‚āā donates electron pair ‚Üí **Lewis base**
- Fe¬≤‚Āļ accepts electron pair ‚Üí **Lewis acid**

## Conjugate Acid-Base Pairs

**Conjugate base** = acid minus H‚Āļ
**Conjugate acid** = base plus H‚Āļ

```
NH‚āÉ + H‚āāO ‚áĆ NH‚āĄ‚Āļ + OH‚ĀĽ
          ‚ÜĎ      ‚ÜĎ
        base   acid

          NH‚āĄ‚Āļ = conj. acid of NH‚āÉ
          OH‚ĀĽ = conj. base of H‚āāO

Reverse:  NH‚āĄ‚Āļ = acid, OH‚ĀĽ = base
          NH‚āÉ = conj. base of NH‚āĄ‚Āļ
          H‚āāO = conj. acid of OH‚ĀĽ
```

## Acid/Base Strength

### Acid Dissociation Constant (K<sub>a</sub>)
```
HA + H‚āāO ‚áĆ H‚āÉO‚Āļ + A‚ĀĽ
K‚āź = [H‚āÉO‚Āļ][A‚ĀĽ] / [HA]
```
- Larger K<sub>a</sub> ‚Üí stronger acid
- pK<sub>a</sub> = ‚Äďlog K<sub>a</sub>
- **Lower pK<sub>a</sub> ‚Üí stronger acid**

### Base Dissociation Constant (K<sub>b</sub>)
```
B + H‚āāO ‚áĆ HB‚Āļ + OH‚ĀĽ
K_b = [HB‚Āļ][OH‚ĀĽ] / [B]
```
- Larger K<sub>b</sub> ‚Üí stronger base
- pK<sub>b</sub> = ‚Äďlog K<sub>b</sub>
- **Lower pK<sub>b</sub> ‚Üí stronger base**

## Amphoteric Substances
- Can act as **both** acid and base
- Conjugate base of a weak polyprotic acid is always amphoteric
- Example: HCO‚āÉ‚ĀĽ (bicarbonate)
  - As base: HCO‚āÉ‚ĀĽ + H‚āāO ‚áĆ H‚āāCO‚āÉ + OH‚ĀĽ
  - As acid: HCO‚āÉ‚ĀĽ + H‚āāO ‚áĆ CO‚āÉ¬≤‚ĀĽ + H‚āÉO‚Āļ
- **Amino acids** are all amphoteric

## Buffer Solutions

**Buffer** = resists pH change when small amounts of acid/base added
- Contains weak acid + conjugate base (or weak base + conjugate acid)
- In roughly equal concentrations

### Bicarbonate Buffer System (Most important in blood)
```
Reaction 1:  H‚āāCO‚āÉ ‚áĆ H‚Āļ + HCO‚āÉ‚ĀĽ
Reaction 2:  CO‚āā + H‚āāO ‚áĆ H‚āāCO‚āÉ
```

**During exercise:** Muscles produce lactic acid ‚Üí H‚Āļ increases
- Reaction 1 shifts LEFT (Le Ch√Ętelier's principle)
- Free H‚Āļ reduced ‚Üí pH drop minimized

**Holding breath:** CO‚āā accumulates ‚Üí Reaction 2 shifts RIGHT ‚Üí more H‚āāCO‚āÉ ‚Üí more H‚Āļ ‚Üí **pH decreases**

## Quick Reference

| Property | Stronger Acid | Stronger Base |
|----------|---------------|---------------|
| K value | Larger K<sub>a</sub> | Larger K<sub>b</sub> |
| pK value | **Lower** pK<sub>a</sub> | **Lower** pK<sub>b</sub> |
| pH | Lower | Higher |

**Examples:**
- Lactic acid (pK<sub>a</sub> = 3.9) > Uric acid (pK<sub>a</sub> = 5.6)
- Lowest [H‚āÉO‚Āļ] = highest pH (seawater pH 8.5)
- Least acidic = highest pK<sub>a</sub> (bicarbonate pK<sub>a</sub> = 10.33)
