# Acids and Bases

## Definitions

### Brønsted-Lowry (Most important for MCAT)
- **Acid** = proton (H⁺) donor
- **Base** = proton (H⁺) acceptor
- Any anion or neutral species with a lone pair can be a base

**Example:**
```
H₂CO₃ + H₂O ⇌ H₃O⁺ + HCO₃⁻
Acids: H₂CO₃, H₃O⁺
Bases: HCO₃⁻, H₂O
```

### Lewis (Broader)
- **Acid** = electron-pair acceptor
- **Base** = electron-pair donor
- Forms coordinate covalent bonds

**Example (heme group):** O₂ binds to Fe²⁺
- O₂ donates electron pair → **Lewis base**
- Fe²⁺ accepts electron pair → **Lewis acid**

## Conjugate Acid-Base Pairs

**Conjugate base** = acid minus H⁺
**Conjugate acid** = base plus H⁺

```
NH₃ + H₂O ⇌ NH₄⁺ + OH⁻
          ↑      ↑
        base   acid
        
Forward:  NH₃ = base, H₂O = acid
          NH₄⁺ = conj. acid of NH₃
          OH⁻ = conj. base of H₂O

Reverse:  NH₄⁺ = acid, OH⁻ = base
          NH₃ = conj. base of NH₄⁺
          H₂O = conj. acid of OH⁻
```

## Acid/Base Strength

### Acid Dissociation Constant (K<sub>a</sub>)
```
HA + H₂O ⇌ H₃O⁺ + A⁻
Kₐ = [H₃O⁺][A⁻] / [HA]
```
- Larger K<sub>a</sub> → stronger acid
- pK<sub>a</sub> = –log K<sub>a</sub>
- **Lower pK<sub>a</sub> → stronger acid**

### Base Dissociation Constant (K<sub>b</sub>)
```
B + H₂O ⇌ HB⁺ + OH⁻
K_b = [HB⁺][OH⁻] / [B]
```
- Larger K<sub>b</sub> → stronger base
- pK<sub>b</sub> = –log K<sub>b</sub>
- **Lower pK<sub>b</sub> → stronger base**

## Amphoteric Substances
- Can act as **both** acid and base
- Conjugate base of a weak polyprotic acid is always amphoteric
- Example: HCO₃⁻ (bicarbonate)
  - As base: HCO₃⁻ + H₂O ⇌ H₂CO₃ + OH⁻
  - As acid: HCO₃⁻ + H₂O ⇌ CO₃²⁻ + H₃O⁺
- **Amino acids** are all amphoteric

## pH and pOH

**pH = –log[H⁺]** (or –log[H₃O⁺])
- [H⁺] = 10⁻ᵖᴴ
- Pure water: [H⁺] = 10⁻⁷ M → pH = 7 (neutral at 25°C)

| pH Range | Solution |
|----------|----------|
| < 7 | Acidic |
| = 7 | Neutral |
| > 7 | Basic |

**pOH = –log[OH⁻]**
- [OH⁻] = 10⁻ᵖᴼᴴ
- At 25°C: **pH + pOH = 14**

## Buffer Solutions

**Buffer** = resists pH change when small amounts of acid/base added
- Contains weak acid + conjugate base (or weak base + conjugate acid)
- In roughly equal concentrations

### Bicarbonate Buffer System (Most important in blood)
```
Reaction 1:  H₂CO₃ ⇌ H⁺ + HCO₃⁻
Reaction 2:  CO₂ + H₂O ⇌ H₂CO₃
```

**During exercise:** Muscles produce lactic acid → H⁺ increases
- Reaction 1 shifts LEFT (Le Châtelier's principle)
- Free H⁺ reduced → pH drop minimized

**Holding breath:** CO₂ accumulates → Reaction 2 shifts RIGHT → more H₂CO₃ → more H⁺ → **pH decreases**

## Quick Reference

| Property | Stronger Acid | Stronger Base |
|----------|---------------|---------------|
| K value | Larger K<sub>a</sub> | Larger K<sub>b</sub> |
| pK value | **Lower** pK<sub>a</sub> | **Lower** pK<sub>b</sub> |
| pH | Lower | Higher |

**Examples:**
- Lactic acid (pK<sub>a</sub> = 3.9) > Uric acid (pK<sub>a</sub> = 5.6)
- Lowest [H₃O⁺] = highest pH (seawater pH 8.5)
- Least acidic = highest pK<sub>a</sub> (bicarbonate pK<sub>a</sub> = 10.33)
