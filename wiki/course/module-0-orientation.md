## 🧠 How to Learn Effectively (Evidence-Based)

### **1. Active Recall > Passive Re-reading**
| Passive (Low Retention) | Active (High Retention) |
|-------------------------|-------------------------|
| Re-reading notes | Close book → explain aloud |
| Highlighting | Write questions, answer later |
| Watching videos | Work problems *before* reading solution |
| Re-watching lectures | Draw concept maps from memory |

**Rule:** If it feels easy, you're probably not learning. Struggle = learning.

### **2. The Feynman Technique (Use for Every Concept)**
```
1. Pick a concept (e.g., "ΔG = ΔH - TΔS")
2. Explain it to a 5-year-old in 2-3 sentences
3. Identify where you got stuck → that's your gap
4. Re-read *only that gap* → simplify explanation
5. Repeat until smooth
```

### **3. Spaced Repetition Intervals**
| Review # | Interval | What to Review |
|----------|----------|----------------|
| 1 | Same day | Concept checks + wrong practice problems |
| 2 | 1 day later | Summary sheets + missed problems |
| 3 | 3 days later | Full module summary + cross-links |
| 4 | 1 week | Mastery check problems |
| 5 | 2 weeks | Full mastery check |

**Tool suggestion:** Anki deck with these cards:
- Front: Equation/concept name → Back: Equation + when to use + common trap
- Front: Concept name → Back: Feynman explanation + MCAT trap

### **4. Cross-Linking Strategy (The MCAT Secret Weapon)**
The MCAT tests **connections**, not isolated facts. Every module has 🔗 **Cross-Links**. When you see one:

```
🔗 CROSS-LINK: [Thermodynamics] ΔG = ΔG°' + RT ln Q
→ Connects to: [Acids/Bases] pH affects Q via [H⁺]
→ Connects to: [Redox] E°' relates to ΔG°' via ΔG°' = -nFE°'
→ MCAT loves: "How does pH change affect ΔG of this redox reaction?"
```

**Your job:** When you hit a cross-link, STOP. Draw the connection. Write one integrated question.
